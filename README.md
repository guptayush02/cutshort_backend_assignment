# clone project
`git clone git@github.com:guptayush02/cutshort_backend_assignment.git`

# install node_modules
`npm install`

# create env
`create a file .env and copy config from .env.example`

# build project
`npm run build`

# start project
`npm start`

# run test
`npx jest tests/<PATH_TO_TEST_FILE>`
